Um criador de fichas de RPG automático para o sistema de Tormenta20.

Sobre o Projeto

Plataforma web completa para a criação, edição e gerenciamento de fichas de personagens para o sistema de RPG Tormenta20. O projeto visa facilitar a vida dos jogadores e mestres, automatizando cálculos complexos e permitindo o armazenamento seguro das fichas na nuvem.

Com uma interface intuitiva e responsiva, os usuários podem criar personagens passo a passo, gerenciar inventários, magias e habilidades, além de exportar as fichas quando necessário.
