Monstros e Máquinas

Modificação para Minecraft com novas mecânicas e itens 3D.

Sobre o Projeto

"Monstros e Máquinas" é um projeto de modding para Minecraft que visa enriquecer a experiência de jogo através da adição de novos itens, armaduras com modelos 3D personalizados e mecânicas de gameplay inovadoras.

O desenvolvimento envolve o uso da linguagem Java com a API NeoForge (versão 1.21.1), integrando bibliotecas como GeckoLib para animações e modelos complexos criados no Blockbench. O projeto demonstra habilidades em programação de jogos, manipulação de assets 3D e lógica de modding.

GitHub Repo

Tecnologias Utilizadas

  • Linguagem: Java
  • API/Framework: NeoForge 1.21.1
  • Modelagem 3D: Blockbench
  • Bibliotecas: GeckoLib, KubeJS
